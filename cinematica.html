Cinemática Geométrica

La cinemática permite determinar los ángulos de las articulaciones del robot a partir de una posición en el espacio.

Concepto básico

Se definen coordenadas X, Y, Z y se calculan los ángulos del robot.

Ejemplo

x = 10;
y = 5;
z = 3;

theta1 = atan2(y, x);

Relación con el robot

  • θ1 → rotación base
  • θ2 → brazo
  • θ3 → antebrazo
  • θ4 → pinza
